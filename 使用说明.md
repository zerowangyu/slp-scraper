# Shopify 商品数据爬虫

通用 Shopify 网站商品数据采集工具，支持任意 Shopify 网站。

## 功能特点

- ✅ 支持任意 Shopify 网站
- ✅ 自动检测网站类型
- ✅ 获取完整商品数据
- ✅ 支持分页采集
- ✅ 自动去重
- ✅ 导出 CSV 文件

## 下载

从 [Releases](https://github.com/zerowangyu/slp-scraper/releases) 页面下载：

- **Windows**: `Shopify商品爬虫-Windows.exe`
- **macOS**: `Shopify商品爬虫-macOS`

## 使用方法

### Windows

1. 下载 `Shopify商品爬虫-Windows.exe`
2. 双击运行
3. 输入要爬取的网站地址
4. 等待完成，CSV 文件会保存在同目录

### macOS

1. 下载 `Shopify商品爬虫-macOS`
2. 打开终端，运行：
```bash
chmod +x Shopify商品爬虫-macOS
./Shopify商品爬虫-macOS
```
3. 输入要爬取的网站地址
4. 等待完成，CSV 文件会保存在同目录

## 采集的数据

| 字段 | 说明 |
|------|------|
| product_name | 商品名称 |
| sku | SKU 编码 |
| barcode | 条形码 |
| category | 分类 |
| vendor | 品牌/供应商 |
| product_type | 商品类型 |
| price | 价格 |
| compare_at_price | 原价 |
| stock_status | 库存状态 |
| product_url | 商品链接 |

## 常见问题

### Q: 提示"不是 Shopify 网站"？

确保输入的是 Shopify 搭建的网站。可以通过访问 `网站地址/products.json` 来验证。

### Q: 运行很慢？

为避免被限流，程序设置了请求间隔，这是正常的。

### Q: CSV 乱码？

用 Excel 打开时选择 UTF-8 编码，或直接拖入 Excel。
