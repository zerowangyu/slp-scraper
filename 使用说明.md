# SLP Trading 商品数据爬虫 - 使用说明

## 功能介绍

本工具可以自动从 slptrading.com.au 网站获取所有商品信息，并保存为 CSV 文件。

**获取的数据包括：**
- 商品名称 (product_name)
- SKU编码 (sku)
- 分类 (category)
- 价格 (price)
- 原价/比较价 (compare_at_price)
- 商品链接 (product_url)

---

## 方式一：直接使用打包好的程序（推荐）

### Windows 用户

1. 下载 `SLP商品爬虫.exe` 文件
2. 双击运行
3. 程序会自动开始爬取数据，等待完成即可
4. 完成后，在 exe 文件所在目录会生成 `slp_products.csv` 文件
5. 用 Excel 打开 CSV 文件查看数据

### 注意事项

- 首次运行时，Windows 可能会弹出安全警告，点击「更多信息」→「仍要运行」即可
- 运行过程中请保持网络连接
- 程序运行需要几分钟，请耐心等待
- 运行结束后会显示"爬虫任务完成！"，按任意键关闭窗口

## 方式二：从源码打包 EXE 文件

如果你需要自己打包 exe 文件（比如修改了代码后），请按以下步骤操作：

### 步骤 1：安装 Python

1. 访问 https://www.python.org/downloads/
2. 下载最新版 Python（建议 3.10 或以上）
3. 安装时**务必勾选** "Add Python to PATH"

### 步骤 2：安装依赖

打开命令提示符（按 `Win + R`，输入 `cmd`，回车），依次执行：

```bash
cd 你的文件夹路径
pip install -r requirements.txt
pip install pyinstaller
```

### 步骤 3：打包为 EXE

在命令提示符中执行：

```bash
python build_exe.py
```

或者手动执行：

```bash
pyinstaller --onefile --console --name "SLP商品爬虫" slp_scraper.py
```

打包完成后，`dist` 文件夹中会生成 `SLP商品爬虫.exe`。

## 常见问题

### Q: 运行时提示"Windows 已保护你的电脑"怎么办？

这是 Windows SmartScreen 的安全提示，因为程序没有数字签名。

**解决方法：** 点击「更多信息」→「仍要运行」

### Q: 运行后没有生成 CSV 文件？

1. 检查是否有网络连接
2. 查看程序窗口是否有错误提示
3. 确认程序运行完成（显示"爬虫任务完成"）

### Q: CSV 文件用 Excel 打开是乱码？

CSV 文件使用 UTF-8 编码，Excel 打开方法：

1. 打开 Excel → 数据 → 从文本/CSV
2. 选择文件，编码选择「UTF-8」
3. 点击加载

或者直接将 csv 文件拖入 Excel，一般都能正确显示。

### Q: 程序运行很慢？

为了避免被网站限制访问，程序设置了请求间隔。这是正常的，请耐心等待。

## 输出文件说明

程序会生成 `slp_products.csv` 文件，包含以下字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| product_name | 商品名称 | Example Product - Size M |
| sku | SKU编码 | ABC-123 |
| category | 所属分类 | Electronics; Accessories |
| price | 当前价格 | 29.99 |
| compare_at_price | 原价（划线价） | 39.99 |
| product_url | 商品页面链接 | https://slptrading.com.au/products/xxx |
